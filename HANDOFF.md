# Хэндофф: состояние проекта и TODO до прода

Документ для этапа полировки. Кратко: что готово, что нужно от коллег, что нужно от тебя, и как всё запустить локально.

## 1. Что готово (в коде, проверено на живом стеке — 30/30 тестов)

- **Отчёты** `/research/:slug` — 8 аналитических отчётов (реальные данные: Федресурс, РБК, «Ведомости», ПравоТех), графики (recharts), PDF в `public/reports/`. Управление в админке (вкладка «Отчёты»).
- **Чек-листы** `/checklists/:slug` — 4 интерактивных чек-листа по 127-ФЗ (галочки, прогресс, сохранение в браузере). Админка — вкладка «Чек-листы».
- **Лендинг платформы** `/platform` — продуктовый лендинг HunterLite (нарратив, мокапы тренировки и дашборда). Промо-блок на главной. Ссылка на сервис — настройка `platform_url`.
- **Лендинг книги** `/book` — полноценный лендинг с формой получения. Контент в админке (поля `book_*`).
- **Реклама АУ** + **подписка на Telegram** — блоки на главной, тексты/ссылки в админке.
- **SEO** — `npm run build` пререндерит 16 статических страниц с индивидуальными OG/canonical, генерит `sitemap.xml` и `robots.txt`, есть `og-cover.png`.
- **Безопасность** — `requireAuth` на всех админ-роутах и чтении лидов (PII); защита от SQL-инъекций (белый список колонок); rate-limit (10/мин/IP) на публичных формах; `nosniff`; `trust proxy`.
- **Инфраструктура** — `infra/docker-compose.yml` (локальный Postgres), `.env.example`, README с локальным запуском.

## 2. Материалы от коллег — статус

- **Влад (Telegram): ПОЛУЧЕНО.** Канал `t.me/kredit_advokat` подключён по умолчанию. Сделана **живая лента постов**: сервер тянет публичную ленту `t.me/s/kredit_advokat` (без API-ключей, кэш 5 мин) и рендерит последние посты на сайте + кнопка «Подписаться». Канал можно сменить в Админке → Настройки → «Подписка на Telegram». ВАЖНО: ленте нужен исходящий интернет с сервера к `t.me` — на localhost/проде есть; если t.me недоступен, блок просто показывает кнопку подписки без ленты.
- **Вася (книга): ПОЛУЧЕНО.** Книга «Банкротство физических лиц» (ред. май 2026) собрана в лендинг `/book`: реальные название, описание, структура из 4 частей и 12 глав; PDF (117 стр.) сконвертирован и лежит в `public/book/bankrotstvo-fizlic.pdf`, скачивается по форме. Обложку можно добавить в Админке (поле «Книга по БФЛ»).
- **Егор (платформа): pre-prod, сервера пока нет.** Лендинг `/platform` готов и показывает «Скоро». Когда Егор развернёт сервис — вписать его URL в Админке (`platform_url`), и кнопки «Открыть платформу» оживут. Развёртывание самой платформы — из репозитория hunterlite (сервер + PostgreSQL + секрет сессии + опц. OAuth-ключи).

## 5. Локальный запуск (для проверки без прода)

```sh
npm install
cp .env.example .env
docker compose -f infra/docker-compose.yml up -d
npm --prefix server run migrate
npm run dev
```

Сайт — `http://localhost:8080`, API — `:3001`, админка — `/admin` (пароль `ADMIN_PASSWORD` из `.env`).
